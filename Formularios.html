Guía de Formularios HTML

Los formularios HTML permiten a los usuarios interactuar con una página web mediante el envío de datos, ya sea a través de texto, botones de opción, casillas de verificación, y otros elementos de entrada.

Elementos Básicos del Formulario

Un formulario en HTML se define con la etiqueta <form>. Dentro de esta etiqueta, puedes incluir varios tipos de entradas como <input>, <textarea>, y <select>.

<form action="/ruta/del/servidor" method="POST">
    ... 
</form>

Entradas de Texto

Las entradas de texto permiten a los usuarios ingresar información libremente en un campo de texto.

Campo de Texto

La entrada más común es un campo de texto, que se crea con <input type="text">.

<label for="nombre">Nombre:</label>
<input type="text" id="nombre" name="nombre">

Campo de Contraseña

Para recoger contraseñas, se utiliza <input type="password">, que oculta el texto ingresado.

<label for="contrasena">Contraseña:</label>
<input type="password" id="contrasena" name="contrasena">

Campo de Correo Electrónico

El campo de correo electrónico se define usando <input type="email">, el cual valida automáticamente la sintaxis del correo.

<label for="email">Correo Electrónico:</label>
<input type="email" id="email" name="email">

Área de Texto

Para capturar texto largo, se usa la etiqueta <textarea>.

<label for="mensaje">Mensaje:</label>
<textarea id="mensaje" name="mensaje" rows="4" cols="50"></textarea>

Botones de Opción y Casillas de Verificación

Los botones de opción permiten seleccionar solo una opción entre varias, mientras que las casillas de verificación permiten seleccionar múltiples opciones.

Botones de Opción

Los botones de opción se crean usando <input type="radio">. Todos los botones relacionados deben compartir el mismo nombre.

<label>
    <input type="radio" name="genero" value="masculino"> Masculino
</label>
<label>
    <input type="radio" name="genero" value="femenino"> Femenino
</label>

Casillas de Verificación

Las casillas de verificación se crean usando <input type="checkbox">.

<label>
    <input type="checkbox" name="intereses" value="musica"> Música
</label>
<label>
    <input type="checkbox" name="intereses" value="deportes"> Deportes
</label>

Listas Desplegables

Las listas desplegables permiten a los usuarios seleccionar una opción de una lista de valores predefinidos. Se crean usando las etiquetas <select> y <option>.

<label for="pais">País:</label>
<select id="pais" name="pais">
    <option value="mexico">México</option>
    <option value="espana">España</option>
    <option value="argentina">Argentina</option>
</select>

Botón de Envío

El botón de envío se define con <input type="submit">. Este botón envía los datos del formulario al servidor especificado en el atributo action del formulario.

<input type="submit" value="Enviar">

Conclusión

Los formularios HTML son una herramienta esencial para recopilar datos de los usuarios en la web. Esta guía cubre los elementos fundamentales que se pueden utilizar en un formulario para capturar diferentes tipos de datos.
