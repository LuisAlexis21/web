Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Fue desarrollado por Tim Berners-Lee en 1991, y se utiliza para estructurar el contenido en la web, creando textos, imágenes, enlaces y otros elementos interactivos.

Estructura Básica de un Documento HTML

Todo documento HTML sigue una estructura básica que consiste en etiquetas (tags) que definen distintos elementos del contenido.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la Página</title>
</head>
<body>

    <h1>Encabezado Principal</h1>
    <p>Este es un párrafo de ejemplo.</p>

</body>
</html>

Explicación de la Estructura

  • <!DOCTYPE html>: Define el tipo de documento como HTML5.
  • <html>: La etiqueta raíz que contiene todo el contenido HTML.
  • <head>: Contiene metadatos como el título y las configuraciones de la página.
  • <meta charset="UTF-8">: Define la codificación de caracteres, recomendada como UTF-8 para soportar la mayoría de los caracteres.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Configura la vista para dispositivos móviles.
  • <title>: Define el título que se muestra en la pestaña del navegador.
  • <body>: Contiene el contenido visible de la página, como textos, imágenes, y enlaces.

Encabezados en HTML

Los encabezados en HTML se utilizan para definir la jerarquía de títulos en un documento. Existen seis niveles de encabezados, desde <h1> hasta <h6>, siendo <h1> el más importante.

<h1>Este es un Encabezado de Nivel 1</h1>
<h2>Este es un Encabezado de Nivel 2</h2>
<h3>Este es un Encabezado de Nivel 3</h3>

Los encabezados ayudan a organizar el contenido, facilitando la lectura tanto para los usuarios como para los motores de búsqueda.

Parrafos y Texto en HTML

Los párrafos se definen con la etiqueta <p>. Es posible aplicar estilos y formatos adicionales utilizando etiquetas de texto como <strong> para negrita o <em> para cursiva.

<p>Este es un párrafo de ejemplo. <strong>Este texto está en negrita</strong> y <em>este texto está en cursiva</em>.</p>

Enlaces en HTML

Los enlaces permiten navegar entre diferentes páginas o recursos en la web. Se crean con la etiqueta <a> y el atributo href, que indica la dirección del enlace.

<a href="https://www.ejemplo.com">Visitar Ejemplo.com</a>

Este enlace llevará al usuario a Ejemplo.com.

Imágenes en HTML

Las imágenes se añaden a una página web usando la etiqueta <img> con atributos como src (fuente de la imagen) y alt (texto alternativo para accesibilidad).

<img src="https://via.placeholder.com/150" alt="Texto descriptivo de la imagen">

Listas en HTML

Existen dos tipos principales de listas en HTML: listas ordenadas (<ol>) y listas no ordenadas (<ul>). Cada ítem de la lista se define con la etiqueta <li>.

Lista Ordenada

<ol>
    <li>Primer ítem</li>
    <li>Segundo ítem</li>
    <li>Tercer ítem</li>
</ol>

Esto genera:

  1. Primer ítem
  2. Segundo ítem
  3. Tercer ítem

Lista No Ordenada

<ul>
    <li>Primer ítem</li>
    <li>Segundo ítem</li>
    <li>Tercer ítem</li>
</ul>

Esto genera:

  • Primer ítem
  • Segundo ítem
  • Tercer ítem

Conclusión

Este documento ha cubierto los conceptos fundamentales de HTML, incluyendo la estructura básica de un documento, cómo trabajar con texto, imágenes, enlaces, y listas. Entender estos conceptos es esencial para empezar a crear páginas web efectivas y accesibles.
