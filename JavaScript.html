Guía de JavaScript

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web. Es uno de los tres lenguajes principales del desarrollo web junto con HTML y CSS.

Cómo Integrar JavaScript en HTML

Hay varias formas de integrar JavaScript en un documento HTML:

1. JavaScript en Línea

Se puede agregar código JavaScript directamente dentro de los elementos HTML utilizando el atributo onclick, onmouseover, entre otros.

<button onclick="alert('Hola Mundo!')">Haz clic aquí</button>

2. JavaScript Interno

Se puede incluir JavaScript dentro de una etiqueta <script> en la sección <head> o <body> del documento HTML.

<script>
    function mostrarMensaje() {
        alert('Este es un mensaje desde un script interno');
    }
</script>

<button onclick="mostrarMensaje()">Mostrar Mensaje</button>

3. JavaScript Externo

Se puede vincular un archivo JavaScript externo usando la etiqueta <script> con el atributo src.

<!-- En el archivo HTML -->
<script src="script.js"></script>

<!-- En el archivo script.js -->
function cambiarTexto() {
    document.getElementById('miTexto').innerHTML = 'El texto ha cambiado!';
}

Variables y Tipos de Datos

En JavaScript, las variables se pueden declarar usando var, let o const. Los tipos de datos comunes incluyen number, string, boolean, entre otros.

let nombre = 'Juan';
let edad = 25;
let esEstudiante = true;

console.log('Nombre:', nombre);
console.log('Edad:', edad);
console.log('Es estudiante:', esEstudiante);

Funciones

Las funciones en JavaScript son bloques de código reutilizables que se pueden definir una vez y ejecutar en cualquier momento.

function saludar(nombre) {
    return 'Hola, ' + nombre + '!';
}

console.log(saludar('María'));

Manipulación del DOM

JavaScript permite acceder y manipular los elementos del DOM (Document Object Model) de una página web.

function cambiarColor() {
    document.body.style.backgroundColor = '#E0F7FA';
}

Eventos

Los eventos en JavaScript permiten que una página web responda a interacciones del usuario, como clics, movimientos del mouse, presiones de teclas, etc.

document.getElementById('miBoton').addEventListener('click', function() {
    alert('Botón clicado!');
});

Conclusión

JavaScript es una herramienta esencial para agregar interactividad y dinamismo a las páginas web. Esta guía ofrece una introducción a los conceptos básicos que cualquier desarrollador web debe conocer.
