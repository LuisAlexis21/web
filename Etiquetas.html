Guía de Etiquetas HTML

HTML está compuesto por diversas etiquetas (o tags) que definen la estructura y el contenido de una página web. Cada etiqueta tiene un propósito específico y es fundamental entender cómo y cuándo usarlas.

Etiquetas Estructurales

Las etiquetas estructurales organizan el contenido en secciones que ayudan a definir la disposición de la página.

<html>

La etiqueta <html> encierra todo el contenido de una página HTML. Es la etiqueta raíz del documento.

<html>
    ...
</html>

<head>

La etiqueta <head> contiene metadatos sobre la página, como el título, enlaces a hojas de estilo, y scripts.

<head>
    <title>Mi Página Web</title>
</head>

<body>

La etiqueta <body> envuelve todo el contenido visible de la página, como textos, imágenes, y enlaces.

<body>
    <h1>Título de la Página</h1>
    <p>Este es un párrafo.</p>
</body>

Etiquetas de Texto

Las etiquetas de texto son utilizadas para dar formato y estructura al contenido textual.

<p> - Párrafo

La etiqueta <p> se utiliza para definir párrafos de texto.

<p>Este es un párrafo.</p>

<h1> a <h6> - Encabezados

Los encabezados van desde <h1> (el más importante) hasta <h6> (el menos importante).

<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>

<strong> y <em> - Negrita y Cursiva

La etiqueta <strong> se utiliza para enfatizar texto en negrita, y <em> para hacerlo en cursiva.

<p>Este es un <strong>texto importante</strong> y este es un <em>texto en cursiva</em>.</p>

Etiquetas de Enlaces

Los enlaces son una parte fundamental de la navegación web, y se crean usando la etiqueta <a>.

<a> - Enlace

La etiqueta <a> define un hipervínculo. El atributo href especifica la URL del enlace.

<a href="https://www.ejemplo.com">Visitar Ejemplo</a>

Etiquetas de Imágenes

Las imágenes se insertan en las páginas web con la etiqueta <img>.

<img> - Imagen

La etiqueta <img> incluye los atributos src (fuente de la imagen) y alt (texto alternativo).

<img src="https://via.placeholder.com/150" alt="Ejemplo de Imagen">

Etiquetas de Listas

Las listas organizan ítems en un formato ordenado o desordenado.

<ul> - Lista No Ordenada

Las listas no ordenadas usan <ul> y se componen de elementos <li>.

<ul>
    <li>Ítem 1</li>
    <li>Ítem 2</li>
    <li>Ítem 3</li>
</ul>
  • Ítem 1
  • Ítem 2
  • Ítem 3

<ol> - Lista Ordenada

Las listas ordenadas usan <ol> y también se componen de elementos <li>.

<ol>
    <li>Primer ítem</li>
    <li>Segundo ítem</li>
    <li>Tercer ítem</li>
</ol>
  1. Primer ítem
  2. Segundo ítem
  3. Tercer ítem

Etiquetas de División y Contenedor

Las etiquetas de división y contenedor permiten agrupar y organizar elementos dentro de la página.

<div> - División

La etiqueta <div> se utiliza para agrupar bloques de contenido, siendo esencial para la estructura de diseño CSS.

<div>
    <p>Este es un bloque de contenido.</p>
</div>

<span> - Contenedor en Línea

La etiqueta <span> es similar a <div>, pero se utiliza para agrupar elementos en línea, como texto dentro de un párrafo.

<p>Este texto tiene <span>una parte resaltada</span> en línea.</p>

Conclusión

Conocer y comprender el uso de las etiquetas HTML es fundamental para estructurar correctamente una página web. Esta guía cubre las etiquetas más comunes que se utilizan en el desarrollo web básico.
