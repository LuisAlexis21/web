Guía de CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento HTML. CSS controla el diseño, la apariencia y la disposición de los elementos en una página web.

Cómo Aplicar CSS

Hay tres formas principales de aplicar CSS a un documento HTML:

1. CSS en línea

El CSS en línea se aplica directamente a los elementos individuales utilizando el atributo style.

<p style="color: blue; font-size: 20px;">Este es un párrafo estilizado en línea.</p>

2. CSS Interno

El CSS interno se coloca dentro de la etiqueta <style> en la sección <head> de un documento HTML.

<style>
    p {
        color: red;
        font-size: 18px;
    }
</style>
<p>Este es un párrafo estilizado internamente.</p>

3. CSS Externo

El CSS externo se define en un archivo separado con la extensión .css. Se vincula al documento HTML usando la etiqueta <link>.

<!-- En el archivo HTML -->
<link rel="stylesheet" href="styles.css">

<!-- En el archivo styles.css -->
p {
    color: green;
    font-size: 16px;
}

Selección de Elementos

Para aplicar estilos a elementos HTML, se utilizan selectores. Algunos de los más comunes son:

Selector de Etiqueta

Este selector aplica estilos a todos los elementos de una determinada etiqueta.

h1 {
    color: #FF5722;
}

Este es un encabezado h1 estilizado

Selector de Clase

El selector de clase se aplica a elementos que tienen un atributo class específico.

.miClase {
    font-style: italic;
    color: #4CAF50;
}

Este párrafo está estilizado con una clase.

Selector de ID

El selector de ID se aplica a un único elemento que tiene un atributo id específico.

#miID {
    text-align: center;
    font-weight: bold;
}

Este párrafo está estilizado con un ID.

Propiedades Comunes de CSS

A continuación, se describen algunas de las propiedades CSS más utilizadas:

Color

Define el color del texto. Se puede especificar en nombre, hexadecimal, RGB, o HSL.

color: #FF0000; /* Rojo */

Background-Color

Define el color de fondo de un elemento.

background-color: #E0E0E0;

Font-Family

Define la fuente utilizada para el texto.

font-family: Arial, sans-serif;

Font-Size

Define el tamaño del texto.

font-size: 18px;

Margin

Define el espacio externo alrededor de un elemento.

margin: 20px;

Padding

Define el espacio interno dentro de un elemento.

padding: 10px;

Border

Define el borde alrededor de un elemento.

border: 1px solid #000000;

Modelo de Caja

El modelo de caja en CSS describe el diseño y el comportamiento del espacio ocupado por un elemento en la página web. Incluye margin, border, padding y content.

div {
    width: 200px;
    padding: 10px;
    border: 5px solid #000000;
    margin: 20px;
}

En este ejemplo, el elemento div tendrá un ancho total de 250px (200px + 10px de padding a cada lado + 5px de borde a cada lado).

Ejemplo Completo de Estilos CSS

A continuación, se muestra un ejemplo completo de cómo combinar varias propiedades CSS para estilizar una página web.

body {
    background-color: #F5F5F5;
    font-family: 'Arial', sans-serif;
}

h1 {
    color: #1976D2;
    text-align: center;
}

p {
    color: #333;
    line-height: 1.6;
    margin: 20px 0;
}

.container {
    max-width: 900px;
    margin: 0 auto;
    padding: 20px;
    background-color: #FFFFFF;
    border-radius: 8px;
    box-shadow: 0 0 10px rgba(0, 0, 0, 0.1);
}

Conclusión

CSS es una herramienta poderosa para mejorar la presentación y el diseño de las páginas web. Esta guía cubre las bases del CSS y cómo aplicarlo para estilizar elementos HTML.
